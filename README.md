# Generative AI and LLM Projects

This repository contains a collection of hands-on Generative AI projects and notebooks covering modern Large Language Model concepts such as RAG, GraphRAG, AI Agents, AutoGen multi-agent systems, DeepEval-based LLM evaluation, PEFT fine-tuning, and building a character-level GPT model from scratch.

The goal of this repository is to demonstrate practical understanding of LLM application development, evaluation, fine-tuning, retrieval systems, and agentic AI workflows.
